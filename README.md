# Skill: Indonesia EYD Writing

## Deskripsi
Skill ini memandu Claude dalam menulis konten yang mematuhi Ejaan Yang Disempurnakan (EYD V) dari Badan Pengembangan dan Pembinaan Bahasa, Kementerian Pendidikan Dasar dan Menengah. Setiap topik memiliki file .md terpisah dengan konten langsung dari situs resmi EYD V (tanpa parafrase).

**Sumber Resmi:** https://ejaan.kemendikdasmen.go.id/eyd/

## Kapan Gunakan
Trigger skill ini ketika:
- User meminta menulis artikel, blog, konten dalam bahasa Indonesia
- User ingin memastikan penulisan mengikuti aturan EYD V
- Ada pertanyaan tentang penggunaan huruf, kata, tanda baca, atau unsur serapan
- Untuk koreksi atau penyempurnaan teks bahasa Indonesia

## Cara Penggunaan

Saat user menulis konten dalam bahasa Indonesia:
1. Lihat file .md yang relevan sesuai pertanyaan mereka
2. Aplikasikan aturan yang tercantum dalam file
3. Berikan contoh dari file tersebut jika diperlukan
4. Jika ada pertanyaan spesifik, referensikan sumber resmi di file

Contoh:
- User tanya tentang tanda elipsis → buka `tanda-elipsis.md`
- User tanya tentang tanda titik → buka `tanda-titik.md`
- User tanya tentang apostrof → buka `tanda-apostrof.md`
