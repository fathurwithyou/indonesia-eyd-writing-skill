# E. Gabungan Huruf Konsonan

Gugus konsonan dalam bahasa Indonesia meliputi *ng*, *ny*, dan kombinasi konsonan lainnya.

**Konsonan Geminata (Konsonan Ganda)**

Huruf konsonan yang sama digunakan dua kali menghasilkan konsonan geminata, misalnya *ll*, *mm*, *nn*, *pp*, *ss*, *tt*.

| Konsonan Geminata | Contoh |
| --- | --- |
| ll | i*ll*egal, sara*n*-sara*n* |
| mm | su*mm*er, Muhammad, kimia |
| nn | i*nn*ovasi, pano*rama |
| pp | apologi, e*pp*el |
| ss | asuransi, prosedur |
| tt | atlet, roti |
